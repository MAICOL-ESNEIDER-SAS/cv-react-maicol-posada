1. Descripción de la Evidencia

En esta primera evidencia se desarrolló la estructura base del proyecto del currículum digital utilizando React y Vite.
El propósito consistió en crear los componentes iniciales del proyecto, organizar el contenido dentro de la carpeta src/components, integrar todos los elementos en el archivo App.jsx y gestionar los avances mediante Git y GitHub, realizando un commit por cada componente solicitado.

Esta evidencia sirve como la base principal para las actividades siguientes (EV06, EV07 y EV08).

3. Componentes creados en la EV04

Durante esta evidencia se implementaron los siguientes componentes base:

3.1. CabeceraCV.jsx

Contiene los datos principales del encabezado del CV: nombre, cargo, ciudad y número de contacto.

3.2. Perfil.jsx

Incluye un resumen del perfil profesional y una lista inicial de habilidades generales.

3.3. Experiencia.jsx

Presenta cuatro experiencias laborales o proyectos formativos relevantes.

3.4. Educacion.jsx

Incluye información académica como cursos, instituciones y años.

3.5. App.jsx

Es el archivo principal donde se integran todos los componentes anteriores, conformando la primera versión del CV funcional.
