## Descripción

En esta evidencia se construyó la **estructura base del CV digital** usando **React + Vite**.

**Objetivos:**
- Crear componentes iniciales.
- Organizar contenido en `src/components`.
- Integrar todo en `App.jsx`.
- Control de versiones con **Git/GitHub** (commit por componente).

Esta evidencia es la base para EV06, EV07 y EV08.

---

## Configuración del Proyecto

```bash
# 1. Clonar el repositorio
git clone <URL-DE-TU-REPOSITORIO>

# 2. Ingresar al proyecto
cd cv-react-maicol-posada

# 3. Instalar dependencias
npm install

# 4. Ejecutar el servidor
npm run dev
```

Accede en: <http://localhost:5173/>

## Componentes creados

*   **CabeceraCV.jsx** → Datos principales (nombre, cargo, ciudad, contacto).
*   **Perfil.jsx** → Resumen profesional + habilidades.
*   **Experiencia.jsx** → 4 experiencias relevantes.
*   **Educacion.jsx** → Información académica.
*   **App.jsx** → Integración de todos los componentes.
